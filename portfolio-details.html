PROJECTS SUMMARY

Otis has engaged in a range of cybersecurity projects to deepen his expertise. His experience includes using Splunk to create indexes, develop alerts, and generate response action reports. He has worked on playbooks for internal SIEM systems and led automation projects with AWS Incident Manager and AWS Systems Manager. His AWS projects involved creating CIDR blocks, subnets, and health-check scripts using Boto3 and Terraform. Otis has also implemented Zero Trust Architecture (ZTA) with Active Directory on Windows Server 2022 and developed Capture The Flag (CTF) events to enhance Linux skills. His work spans forensic projects, network architecture, containerization, and task automation with Ansible and Terraform. Additionally, he has conducted application security testing with SAST, DAST, and IAST tools and contributed to a Security Operations Center (SOC) where he utilized SIEMs, network monitoring, incident response, and threat analysis.
